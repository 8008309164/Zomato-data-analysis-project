# 🍽️ Zomato Restaurants Data Analysis & Expansion Strategy

## 📌 Project Overview
This project analyzes Zomato restaurant data to provide business insights and suggest expansion strategies.

## 🎯 Objectives
- Identify low competition countries
- Analyze ratings and pricing
- Study customer engagement
- Recommend expansion locations

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Dashboard

## 📊 Key Insights
- India has highest restaurant count
- Emerging markets show growth potential
- Online delivery improves engagement
- Mid-range pricing dominates

## 📁 Files Included
- Excel Dashboard
- PowerPoint Presentation
- Project Report

## 🚀 Conclusion
Data-driven analysis helps identify expansion opportunities.
